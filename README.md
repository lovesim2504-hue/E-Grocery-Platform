# E-Grocery-Platform
 E-Grocery-Platform is a Grocery eCommerce Web Application developed using **PHP, MySQL, AJAX, HTML, CSS, JavaScript, and Bootstrap**. The platform allows customers to browse grocery products, manage shopping carts, place orders, and enjoy a seamless online shopping experience with real-time interactions powered by AJAX.

---

## 📖 Overview

FreshCart is designed to simplify grocery shopping by providing an easy-to-use online platform where users can:

- Browse grocery products
- Search products instantly
- Add products to cart
- Manage orders
- Securely register and log in
- Experience real-time cart updates without page refresh
